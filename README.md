# Inventory & Warehouse Management System (IWMS)

## Overview
[cite_start]The **Inventory & Warehouse Management System (IWMS)** is designed to streamline warehouse operations, including stock management, supplier tracking, and real-time inventory reporting. [cite_start]The system provides a reliable platform for administrators and staff to monitor stock levels, manage batches, and prevent shortages[cite: 18].

## Key Features
* [cite_start]**Stock Management**: Efficient handling of Stock In and Stock Out transactions[cite: 22].
* [cite_start]**Automated Alerts**: Real-time low stock notification system to prevent inventory depletion[cite: 23, 81].
* [cite_start]**Supplier Tracking**: Integrated module for managing supplier registration and linking them to stock records[cite: 25, 141].
* [cite_start]**Advanced Reporting**: Generates inventory valuation reports using FIFO, LIFO, and Weighted Average methods[cite: 154].
* [cite_start]**Batch & Expiry Tracking**: Monitors product batches and expiration dates for better quality control[cite: 24, 119].

## System Architecture
[cite_start]The project utilizes a **Layered Architecture** to maintain a clean separation of concerns[cite: 67]:
* [cite_start]**Presentation Layer**: Web-based UI featuring a dashboard and interactive forms[cite: 71].
* [cite_start]**Business Logic Layer**: Core services including Inventory, Supplier, and Report services[cite: 72, 73].
* [cite_start]**Data Access Layer (DAO)**: Handles direct communication with the database[cite: 74, 75].
* [cite_start]**Database Layer**: Supports relational databases like **MySQL** or **PostgreSQL**[cite: 76].

## Technical Specifications
* [cite_start]**Architecture**: Modular Layered Architecture[cite: 68].
* [cite_start]**Database**: Relational (MySQL/PostgreSQL)[cite: 43].
* [cite_start]**UI/UX**: Dashboard-oriented, responsive layout with role-based access control[cite: 334, 340].
* [cite_start]**Performance**: Optimized for bulk transactions and 99.5% system availability[cite: 29, 32].
